UI/UX Project

This project was developed as part of my Principles of Visual Design (LMC 2720) course, where I created a comprehensive style guide for a reimagined Dave & Buster’s brand identity. The assignment challenged me to apply visual design principles—including typography, color theory, iconography, and layout—to produce a cohesive system that modernizes the brand while retaining its nostalgic 90’s arcade energy. Through this process, I explored how design choices can shape user perception, strengthen brand recognition, and enhance overall user experience across digital and print applications.
